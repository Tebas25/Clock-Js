# JS Analog Clock

Este proyecto es parte de un reto de desarrollo web enfocado en el manejo del DOM y las funciones de tiempo en JavaScript.

## Descripción
El objetivo de este reto es construir un reloj analógico funcional utilizando CSS para el diseño y JavaScript para la lógica de movimiento. El reloj muestra la hora actual local en tiempo real, moviendo las manecillas de segundos, minutos y horas con precisión.

Utiliza transiciones CSS para simular el "tic-tac" mecánico y cálculos matemáticos simples para convertir el tiempo en grados de rotación.

## Tecnologías utilizadas
* **HTML5:** Estructura semántica del reloj.
* **CSS3:** Diseño responsivo, Flexbox y transformaciones (`rotate`, `transform-origin`).
* **JavaScript (ES6):** Manipulación del DOM y objeto `Date` para la lógica del tiempo.

## Instrucciones de instalación y ejecución
Este proyecto no requiere dependencias externas ni instalación de paquetes.

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/Tebas25/Clock-Js
    ```
2.  **Abrir el proyecto:**
    Navega a la carpeta del proyecto y abre el archivo `index.html` en tu navegador web de preferencia (Chrome, Firefox, Edge).
